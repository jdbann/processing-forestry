Go For A Walk

Let’s put this lumberjack through their paces then. Nothing too strenuous though, we’re just going to ask them to walk to a specific place.

Walk this way

You should already be able to handle a person asking for tasks. Now lets look at how we specify a task which can be completed:

{
  "tasks": [
      {
        "type": "walk",
        "id": "anything",
        "x": 10,
        "y": 10
      }
  ]
}

So, a task is an object inside the tasks array. All tasks have a type (in this instance, "walk"), an id (which doesn’t matter right now) and other details related to the particular task.

For a walk task, we specify a set of coordinates to indicate where we would like the person to walk to.

How to succeed

Get your lumberjack to walk to any location. When they successfully arrive, you will have succeeded!
